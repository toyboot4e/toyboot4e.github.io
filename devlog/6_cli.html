1. About

モダンな定番のコマンドラインツールなどをリストしました。

1.1. Background

コマンドラインは機能／ UI を両立していて、ほぼ理想郷だと思います。ただそれは、最新のツールを入れたためでもあるので、リンクを集めてみました。

2. Command line tools

2.1. Replacements

最新のツールを使うと、着色や再帰検索がデフォルトとなり、オプションも整理されます。

feature classic modern

manual

man

tealdeer (tldr)

ripgrep (rg)

最低限、 rg と fd は使った方が良いと思います。本当にいいツールなので。 tree や exa -T の代わりに fd + as-tree のコンボもアリです。

他には、 GNU xargs (gxargs), GNU awk (gawk) や rename コマンドをよく使っています。

2.2. Navigation

z の fish 版を使っています。

2.3. Packages

  • coreutils [1]

    sort, uniq, tr などの Unix フィルタは、継続して使っています。

  • moreutils

    エディタで加工する vi シリーズが便利です。 stdout を加工する vipe, 一括リネームの vidir が含まれています。

2.4. Editors

  • NeoVim (nvim)

    Vim の fork です。動作が軽く、僕はこちらを使っています。

  • Emacs

    Evil を入れると Vim になります。実は、起動以外は NeoVim よりも軽快です。ただ、 Evil 特有の不具合やハマりポイントがあって、かなり調整が難しい印象です。また、言語サーバのサポートはかなり弱いと思います (不具合が普通にあります) 。

2.5. Shells, terminal multiplexers

  • fish

    シェルです。 zsh でもいいかもしれません [2] 。

  • tmux

    ターミナルのインスタンスを複数作り [3] 、 session, window, pane という親子関係で整理してくれます。

2.6. Git

  • delta

    diff が綺麗になります。特に行番号を表示できるのが好みです。

  • hub

    GitHub の非公式 CLI ツールです。リポジトリの作成や URL の取得などに便利です。

  • gh

    GitHub 公式 CLI ツールです。僕はまだ活用できていません。

  • ghq

    リモートリポジトリの管理に便利だそうです……？

2.7. Input (macOS only)

OS が進化したかのように便利になります:

  • Karabiner-Elements

    キー入力を自由にマッピングできます。僕は CapsLock を Ctrl+Shift+Alt+Cmd (hyper key) に変えて、ランチャの hot key としています (skhd 経由) 。

  • skhd

    キー入力に応じて、シェルコマンドを実行してくれます。アプリのランチャとして使っており、 yabai コマンドの呼び出しもしています。

  • yabai

    Tiling window manager です。ウィンドウのレイアウトやリサイズなどをしてくれます。

yabai は HACK の塊のような印象があります。 100 % 想定された通り動作すると期待しない方が良いかもしれません。

2.8. Markup languages

  • Asciidoctor

    表現力豊かな markup 言語 (AsciiDoc) の実装です。このブログ生成にも asciidoctor が使われています。 org-mode も良さそうです。

  • pandoc

    主に markdown, html の相互変換に使っています。

2.9. Downloaders

dl コマンドを作り、与えられた URL に応じて、ダウンローダを割り当てています。

  • youtube-dl

    youtube やニコニコした動画／音源をダウンロードしてくれます。

  • bandcamp-dl

    bandcamp から音源をダウンロードしてくれます。タグもちゃんと付けてくれるのが良いと思います。

  • narou.rb (narou)

    小説家になる感じのサイトなどから原稿をダウンロードし、電子書籍に変換してくれます。 pixiv 小説は非対応。

2.10. Browsers

  • cmus

    音楽のプレイヤで、 Vim 風のキーバインドで動きます。実用しています。

  • w3m

    古いブラウザです。 JS は動きません。これはたまに使います。

  • rtv

    reddit の閲覧ができる CLI ツールです。十分に使用できますが、 web 版の方が便利でした。

  • cordless

    Discord の CLI ツールです。不完全な擬似的 GUI といった印象で、そもそも検索ができません？

  • hnterm

    慣れなくて使っていません。

3. Wrapping up

ターミナルの魅力は、アプリのサンドボックスであることだと思います。 tmux を使えば、思うさま画面分割ができ、文字しか表示されないことは、 コンスタントなレイアウト が与えられる意味でメリットと言えます。しかし、これが最高の環境かと言えば、実はグラフィカルな環境が再び覇権を取り戻すのではないかと予想しています (?) 。

一方で、今日のターミナルは、ほとんど極まっており、今後大きく上回るものが出るかには疑問があります。UI が最高水準に達した後は、人間が進化するしかないのかもしれません。
