1. Books

特定のプログラミング言語に関する本をリストしました。未読の物が多いです。

言語の入門書は、細かい言語仕様から語り始めるものが多い気がします。僕はモジュールの分け方を最初に知りたいので、参考になるリポジトリを探すところが出発点になりがちです。

1.2. Finished reading

The Rust Programming Language

書名 The Rust Programming Language (Covers Rust 2018)

著者

Steve Klabnik

出版日

August 12, 2019

ページ数

560

Rust の公式本です。長い上に内容不足なのですが、よく書かれており、読後は何らかの教育を受けたと感じることができます。そういった感覚が好きなら、読み通すのも良いと思います。

効率に関しては、日本語で書かれた Rust の本を読んだ方が速いかもしれません。また、 Rust 言語そのものを理解するには、別の積み重ねが必要そうです。たとえば、 Oxide: The Essence of Rust を読んでみるとか……。

Rust について書かれた日本語のブログを探すと、凄い人たちばかりで、とても良かったです。現在のこの人に僕は一生及ばないかも、と思わせられる専門性を感じました:

  • Lispエイリアンの狂想曲 | κeenのHappy Hacκing Blog

  • 続くといいな日記

  • 簡潔なQ

などなど……。まだまだ気づいていないブログもたくさんありそうです。
