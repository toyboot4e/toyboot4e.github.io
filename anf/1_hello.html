1. About

Rust から FNA3D を使います。

1.1. Background

僕はゲーム制作に Nez を使っていました。 Nez はオープンソースの 2D フレームワーク (C#) で、 ソースを読めば中身は何でも分かります 。

Nez からの学びは多く、使っていて快適でしたが、むしろ Nez そのものを作りたくなってしまいました。そこで、好みの Rust 言語でフレームワークを作り始めました。最終目標は 2D ローグライクゲームを作る事です。

2. Draw sprites

2.1. FNA

Nez は FNA フレームワークを使っています [1] 。 FNA は XNA の再実装で、 Terraria や Owlboy などのゲームにも使用されています。

今年 FNA のグラフィクス部分が C で書き直されて、 FNA3D というライブラリになりました。 C は ABI (application binary interface) が安定しているため、他のどの言語からも使用できます。

『C ライブラリに依存する』のは、『すべてを Rust で書く』のと比べてクールではないかもしれません。しかし、様々な言語から FNA3D が使用されている状況は面白いと思い、僕もその一員になりたいと思いました。

FNA3D は、 9 月 1 日から FNA の master ブランチで使用される予定です。 7 月 24 日の現在、 API がほぼ安定しているので、使い始めて良いと思います。

Rust-FNA3D

FNA3D を Rust でラッピングしました (Rust-FNA3D):

  • C ライブラリを sub module として追加

    • 自動 pull

  • C ライブラリのビルドの自動化

  • C ライブラリへの FFI bindings の生成 (bindgen)

  • 生成された bindings のラッピング

結構な作業量でした。 Rust-FNA3D を使ったコードは、ユーザから見えない所に隠れるので、ラッピングせずに、 bindgen の出力を直接使っても良かったかもしれません。

Rust から FNA3D を使ったのは、僕が初めてだと思います。

2.2. Sprites = textured quadliterals

それでは絵を出していきます！

Quadliteral = two triangles

FNA3D の描画機能は primitive 単位でした。 Primitive は普通は三角形で、 3 つの vertex (頂点) で指定されます。頂点は、頂点バッファ (頂点配列) への index で指定されます。

四角形は 2 つの三角形であると捉えて、 4 つの頂点と 6 つの index で指定します。

一般の四辺形を quadliteral (quad) と呼ぶようです。四辺形／四角形はよく描くので、 draw_primitives とは別に draw_quads を用意したライブラリもあるかもしれません。

Sprite batch

ゲームは毎フレーム 2 つ (以上) の画面 を切り替えて表示されます。次のフレームに表示する画面のデータを back buffer (back frame buffer) と呼びます。

画像を画面に表示するには、 back buffer への書き込みが必要です。 Back buffer (もしくは任意の render target) への画像データの書き込みは、 GPU 側に依頼します。この依頼コマンドを draw call と呼びます。具体的には、 FNA3D_DrawIndexedPrimitive の呼び出しが draw call に相当します。

GPU はピクセルの計算に強く、 CPU よりも遥かに速く画像処理ができます。しかし、 GPU にデータを送るのは時間がかかります。そこで、同種の (かつ連続した [2]) draw call に関しては、複数データを一括して GPU へ送ることで、効率を良くします (drawcall batching)。

大まかに、同じ画像 (の一部) をどこかへ書き込む命令は、 1 つの draw call に束ねる (batching) 事ができます。

Notes

Why triangles?

三角形を primitive として使うことには、法線が一意的になる、などの妥当な理由があるようです。また、グラフィクスの低レイヤ API (DirectX, OpenGL, Metal, Vulka..) が三角形の使用を前提としています。

3D では、三角形が作ったポリゴン (mesh?) に 2D 画像を貼り付けたします。 2D でも三角形を使うメリット・デメリットなどは気になります。
