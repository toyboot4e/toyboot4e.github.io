=== The Rust Programming Language

[cols="1,3"]
|====
| 書名     | https://www.amazon.co.jp/Steve-Klabnik/dp/1718500440/ref=tmm_pap_swatch_0?_encoding=UTF8&qid=1604043794&sr=1-4[The Rust Programming Language (Covers Rust 2018)]

| 著者     | Steve Klabnik
| 出版日   | August 12, 2019
| ページ数 | 560
|====

{rust} の公式本です。ユーザの観点から書かれています。読むのに時間がかかる上に、この本だけでは実践的なコードは書けません。しかしよく書かれており、読後は何らかの教育を受けたと感じることができます。そういった感覚が好きなら、読み通すのも良いと思います。

効率に関しては、日本語で書かれた Rust の本を読んだ方が速いかもしれません。また、 Rust 言語そのものを理解するには、別の積み重ねが必要そうです。たとえば、 https://arxiv.org/pdf/1903.00982[Oxide: The Essence of Rust] を読んでみるとか……。

Rust について書かれた日本語のブログを探すと、凄い人たちばかりで、とても良かったです。今のこの人に僕は一生及ばないかも、と思わせられる専門性を感じました:

* https://keens.github.io/[Lispエイリアンの狂想曲 | κeenのHappy Hacκing Blog]
* https://mizunashi-mana.github.io/blog/index.html[続くといいな日記]
* https://qnighy.hatenablog.com/[簡潔なQ]

などなど……。まだまだ気づいていないブログもたくさんありそうです。
